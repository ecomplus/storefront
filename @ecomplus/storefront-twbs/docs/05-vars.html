# Usage example

.btn-orange {
  background-color: var(--orange);
  color: var(--white);
}
.y-divisor {
  display: inline-block;
  width: var(--border-width);
  background: var(--secondary);
  height: var(--font-size);
  margin: 0 var(--spacer-3);
}

<button class="btn btn-orange">
  Orange button
</button>
<span class="y-divisor"></span>
<span style="font-weight: var(--font-light); color: var(--text-muted);">
  Light muted text
</span>
Light muted text

# Colors YIQ and RGB

For each brand and context color we also have CSS vars with YIQ contrast color for foregrounds and RGB when you need transparency:

.opaque-info-block {
  background-color: rgba(var(--info-rgb), .85);
  color: var(--info-yiq);
}

<div class="opaque-info-block">
  85% opaque info block
</div>
85% opaque info block

# Brand colors variants

For primary and secondary brand colors we have additional CSS vars with lighten/darken color versions, by default with following weigths:

Label Weigth
whiter 75% lighten
white 50% lighten
lightest 33% lighten
lighter 21% lighten
light 10% lighten
lighten 7% lighten
darken 8% darken
dark 10% darken
darker 13% darken
darkest 16% darken
black 50% darken

Light* and dark* variants also have -yiq and -rgb vars.

You may change those weigths at compilation time with $color-shift-weights Sass map (opens new window) or runtime just overwriting respective CSS vars values with wanted colors.
